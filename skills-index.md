# Skills Index -- Tim (PocketOP-Database)

Consult this file at the start of every session. Load any skills relevant to the task at hand using the Skill tool before proceeding.

## Relevant skills for database and JSON work

- **verification-before-completion** — Use before any commit to run a structured pre-flight check; catches validation gaps before they reach main.
- **verification-loop** — Use when iterating on a JSON change that needs repeated validation until it passes all quality checks.
- **systematic-debugging** — Use when a JSON file fails to parse or an entry has unexpected behaviour; guides root-cause analysis before attempting a fix.
- **json-canvas** — Use when visualising or reasoning about JSON structure, schemas, or relationships between entries.
- **deep-research** — Use when spec data for a new camera or lens entry is incomplete and needs to be sourced from external references.
- **smart-explore** — Use when orienting in the repo at the start of a session or when tracking down where a particular field or pattern is used across files.
- **coding-standards** — Use when adding a new field or entry type to confirm the format matches existing conventions before writing any JSON.
- **make-plan** — Use before tackling a large batch of additions or a schema change that touches multiple files.
